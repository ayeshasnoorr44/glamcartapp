# **App Name**: Glamify

## Core Features:

- AI-Powered Virtual Try-On: Utilize MediaPipe Face Mesh to enable users to virtually try on lipstick and eyeshadow, providing real-time feedback and color accuracy.
- Curated Product Catalog: Showcase a curated list of makeup products from top brands, filtering for those with 4+ star ratings based on verified customer reviews. Allow admins to CRUD new products and filter/search the catalog using brand, price and rating.
- E-Commerce Simulation: Provide shopping cart functionality and checkout simulation for users to add products, manage quantities, and proceed through a simulated purchase process. Users are shown realistic data at checkout.
- Real-Time Analytics: Integrate Microsoft Clarity to track visitor behavior, generate heatmaps, and provide data-driven insights for business optimization, with a comprehensive admin dashboard to visualize the results. This dashboard helps admin refine which products to curate.
- AI product suggestions: Provide personalized product recommendations based on user input and skin type analysis using an AI chatbot tool.

## Style Guidelines:

- Primary color: Deep magenta (#C21E56) to convey sophistication and glamor, mirroring the premium makeup brands.
- Background color: Light pink (#F5E7EB), a very desaturated tone of the primary, to provide a soft, elegant backdrop.
- Accent color: Deep purple (#502CA5) to complement the primary and highlight interactive elements, placed analogously on the color wheel.
- Headline font: 'Playfair', serif. Body font: 'PT Sans', sans-serif. 'Playfair' lends an elegant, high-end feel, well-suited for headlines. Use 'PT Sans' for longer blocks of body text.
- Use elegant, minimalist icons for navigation and product categories.
- Implement a clean, responsive grid layout optimized for both mobile and desktop, following Material Design principles.
- Incorporate subtle, elegant animations and transitions to enhance user engagement.
